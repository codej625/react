# 리액트 명명 규칙을 알아 보자!

<br />

```
1) 공백 X
프로젝트 이름에는 공백이 포함되어서는 안 된다.
공백 대신에 하이픈(-)나 언더스코어(_) 등을 사용할 수 있다.

2) 특수 문자 X
일반적으로 특수 문자는 허용되지 않는다.
오직 알파벳 문자, 숫자, 하이픈(-), 언더스코어만(_)을 사용해야 한다.

3) 숫자로 시작 X
프로젝트 이름은 숫자로 시작할 수 없다.

NPM 규칙 준수 
프로젝트 이름은 NPM 규칙을 따라야 한다.
NPM 규칙에 따르면 소문자만 사용하고,
단일 단어가 아닌 경우에는 대시나 언더스코어를 사용하여 단어를 구분한다.
```
