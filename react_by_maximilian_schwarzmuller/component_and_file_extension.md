# 컴포넌트와 파일 확장자에 대해

<br />

1. jsx?
```
리액트에서 사용하는 .jsx 확장자는 브라우저에서 지원되지 않는 파일 확장자이다.
이것이 작동할 수 있는 이유는 리액트 프로젝트에서 이 특별한 확장자를 지원하기 때문이다.

이 확장자는 개발 서버가 실행될 때,
백그라운드에서 실행되는 빌드 프로세스에게 해당 파일이 JSX 코드를 포함하고 있다는 것을 알려준다.
```
```
jsx 대신 js만 사용하는 리액트 프로젝트도 찾을 수 있다.

그리고 그 .js 파일 안에서도 JSX 코드를 찾을 수 있는데,
이것을 강조하는 이유는 `.jsx` 파일을 사용하지 않는 React 프로젝트를 만났을 때 혼란스럽지 않도록 하기 위함이다.
```
```
일반적으로,
jsx는 리액트 컴포넌트를 정의하는 데 사용.
js는 일반적인 자바스크립트 코드를 작성하는 데 사용.

어떤 확장자를 사용할지는 프로젝트의 설정과 개발자의 선호에 따라 다르다.
```
